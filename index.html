Experience

HUAWEI (Today - August 2022)

Theoretical research in the field of Materialized View Selection Problem. It consists in determining, for a large workload of queries, the optimal set of intermediate computations, the preservation of which will accelerate the execution of the workload. The existing 2018 Microsoft solution has been significantly improved.

HDI lab (May 2022 - September 2021)

Development of a new algorithm for ensembling machine learning models. The optimality of the developed algorithm was proved, as well as resistance to minimization errors. The algorithm was successfully implemented and applied in practice to solve a number of test problems.
